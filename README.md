# Community Marketplace

A simple community marketplace app where users can list items for sale and browse items posted by others.

This README is the **initial starting point** for the project.

---

## Project Idea

The Community Marketplace allows users to:

* Sign up and log in
* Post items for sale
* View items listed by other users

The focus of the first phase is to build a clear and working backend foundation.

---

## Tech Stack

* **Backend:** Node.js + Express
* **Database:** PostgreSQL
* **Authentication:** JWT
* **Image Storage:** Cloudinary
* **API Style:** REST
